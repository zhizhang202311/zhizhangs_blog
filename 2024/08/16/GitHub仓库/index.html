在GitHub上上传项目的步骤如下：

1. 创建一个新的GitHub仓库

  1. 登录你的GitHub账户。
  2. 点击右上角的“+”号，并选择“New repository”。
  3. 填写仓库名称和其他相关信息，然后点击“Create repository”按钮。

    这个目录默认是隐藏的，用 ls -ah 命令就可以看见。

    总结：添加文件到Git仓库，分两步：
    使用命令git add ，注意，可反复多次使用，添加多个文件；
